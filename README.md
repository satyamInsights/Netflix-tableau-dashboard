# 🎬 Netflix Data Analysis Dashboard (Tableau)

## 🚀 Project Overview

This project focuses on analyzing Netflix content using **Tableau** to uncover trends, patterns, and insights from movies and TV shows.

The goal of this project is to practice **data cleaning, visualization, and storytelling** using real-world data.

---

## 📊 Dashboard Features

* 📌 Movies vs TV Shows distribution
* 📈 Content growth over the years
* 🌍 Country-wise content distribution
* 🎭 Genre analysis
* ⭐ Ratings distribution
* 🎯 Interactive filters (Year, Country, Type)

---

## 💡 Key Insights

* Movies dominate Netflix content compared to TV Shows
* Rapid increase in content after 2015
* Majority of content comes from a few countries
* Drama, Comedy, and International shows are most popular

---

## 🛠 Tools Used

* Tableau
* Excel (Data Cleaning)

---

## 📂 Dataset

* Source: Netflix Titles Dataset
* Format: CSV
